List of pages

  1. Workout Tracking Page
    • Here a user will enter their exercises for tracking their workout.
  2. Create User Page
    • Here a user can create their username and profile.
  3. Routine creation/managing.
    • Here a user can create a routine, or manage an existing routine they created.
  4. Routine managing/selection.
    • Here a user can select a routine to follow, and look at other routines available.
  5. Add a musclegroup
    • Here a user can create a musclegroup.
  6. Create an exercise to be used in routines and workout tracking.
    • Here a user can create an exercise that will be used in a routine or for workout tracking.
