List of pages

  1. Workout Tracking Page
    • Here a user will enter their exercises for tracking their workout. They will also have access to a delete and update button for each workout. And will be able to see what exercises they have done and recorded today here.
  2. Display/Update User Page
    • Here a user view and update/delete users.
  3. Create User Page
    • Here a user can create their username and profile.
  4. Routine creation/managing.
    • Here a user can create a routine, or manage an existing routine they created by updating or deleting exercises from the routine.
  5. Routine managing/selection.
    • Here a user can select a routine to follow, and look at other routines available.
  6. Add a musclegroup
    • Here a user can create a musclegroup.
  7. Create an exercise to be used in routines and workout tracking.
    • Here a user can create an exercise that will be used in a routine or for workout tracking.
